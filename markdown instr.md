# Инструкция по работе с markdown

## Выделения текста
Чтобы выделить текст курсивом, надо обрамить его звездочкой (*) Например *вот так*

Чтобы выделить текст полужирным, необходимо обрамить его двумя звездочками (**).  Например, **вот так**
